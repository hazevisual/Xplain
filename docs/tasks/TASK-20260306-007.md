### Scope
- Goal: Implement an in-app "Explain Narrative" module that visually packages agent output directly inside XPlain UI.
- In scope:
  - Narrative API contract and generation endpoint for structured explanation blocks.
  - Frontend narrative panel rendered under/alongside graph results.
  - Styling and layout consistency with current dark minimal UI.
  - Source references and sectioned explanation output (summary, steps, key links/dependencies).
  - Optional export path definition (`HTML/PDF`) as a follow-up phase, not mandatory for first delivery.
- Out of scope:
  - Standalone external HTML workflow requiring manual file opening.
  - Full internet-scale ingestion ("all Wikipedia") or unrestricted crawling.
  - Enterprise RBAC and production auth rollout.
- Constraints:
  - Narrative must be a native part of XPlain application flow.
  - Keep existing process/version/workflow contracts backward compatible.
  - Preserve current UI architecture and block alignment rules.
